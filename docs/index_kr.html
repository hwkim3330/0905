4포트 크레딧 기반 셰이퍼 TSN 스위치 구현 및 성능 평가

차량용 이더넷에서의 QoS 보장

차량 네트워크를 위한 시간 민감형 네트워킹 연구

요약

최근 전기/전자(E/E) 아키텍처가 영역(존) 기반 구조로 진화함에 따라, 차량 내 네트워크는 고대역폭 멀티미디어 스트림과 중요 제어 데이터를 포함한 다양한 트래픽 유형의 신뢰성 있는 전송을 요구받고 있다. 이러한 요구에 대응하기 위해 크레딧 기반 셰이퍼(CBS)를 이용해 시스템을 구현하고, 4포트 차량용 TSN(Time-Sensitive Networking) 스위치를 이용해 시험 및 성능 평가를 수행하였다.

실험 환경으로는 다수의 영상 스트림 소스, 두 개의 영상 수신기, 네트워크 혼잡을 유발하는 베스트 에포트(BE) 트래픽 생성기를 구성하였다. CBS 적용 결과, 네트워크 과부하 상황에서도 시간 민감성 영상 스트림의 우선순위 제어를 통해 안정적인 재생과 최소한의 프레임 손실을 달성하였다.

1. 서론

1.1 연구 배경

현대 차량의 전기/전자(E/E) 아키텍처는 도메인 중심에서 존(Zone) 기반 구조로 급속히 진화하고 있다. 이러한 변화는 자율주행, 첨단 운전자 지원 시스템(ADAS), 인포테인먼트 시스템의 발전과 함께 차량 내 데이터 트래픽이 폭발적으로 증가함에 따른 필연적인 결과이다.

1.2 TSN과 크레딧 기반 셰이퍼

TSN은 IEEE 802.1 워킹 그룹에서 개발한 표준 집합으로, 표준 이더넷 네트워크에서 실시간성과 신뢰성을 보장하기 위한 다양한 메커니즘을 정의한다. IEEE 802.1Qav에 정의된 크레딧 기반 셰이퍼(CBS)는 각 트래픽 클래스에 대해 대역폭을 예약하고 버스트를 제어하여 지연 시간을 예측 가능하게 만든다.

1.3 연구 목적

  • 하드웨어 기반 CBS 구현 및 검증
  • 정량적 성능 평가
  • 차량 네트워크 적용 가능성 검증
  • 구현 가이드라인 제공

2. 크레딧 기반 셰이퍼 이론

2.1 CBS 동작 원리

CBS는 각 트래픽 클래스에 대해 실시간으로 크레딧 값을 관리하여 전송 자격을 결정한다. 큐가 비어있을 때는 idleSlope 속도로 크레딧이 증가하고, 프레임 전송 중에는 sendSlope 속도로 감소한다.

큐가 비어있을 때:

credit_c(t) = credit_c(t-1) + idleSlope × Δt

프레임 전송 중:

credit_c(t) = credit_c(t-1) + sendSlope × Δt

2.2 CBS 파라미터

  • idleSlope: 큐가 유휴 상태일 때 크레딧 증가 속도 (보장 대역폭)
  • sendSlope: 전송 중 크레딧 감소 속도
  • hiCredit: 최대 크레딧 한계
  • loCredit: 최소 크레딧 한계

3. 시스템 구현

3.1 하드웨어 플랫폼

EVB-LAN9692-LM

  • CPU: ARM Cortex-A53 @ 1 GHz
  • 4× SFP+ 포트
  • IEEE 802.1Qav CBS 지원
  • IEEE 802.1Qbv TAS 지원

EVB-LAN9662

  • CPU: ARM Cortex-A7 @ 600 MHz
  • 2× RJ45 기가비트 이더넷
  • 완전한 TSN 기능 세트
  • 실시간 엔진 (RTE)

3.2 트래픽 구성

5. 결론

5.1 주요 발견사항

  • CBS가 idle-slope 구성에 따라 출력 속도를 성공적으로 제어 (100 Mbps)
  • 포트 8에서 제어된 전송으로 적절한 CBS 동작 확인
  • 과부하 조건에서도 출력 포트가 안정적인 전송 속도 유지
  • 포트 10과 11에 대한 추가 구성 세부 조정 필요

5.2 향후 연구

  • 모든 입력 포트에 균일한 CBS 구성 보장
  • 최적화를 위한 다양한 idle-slope 파라미터 테스트
  • Time-Aware Shaper (TAS)와 CBS 통합
  • 실제 차량 멀티미디어 트래픽 패턴에 적용

5.3 산업적 영향

본 연구는 차량 네트워크에서 QoS 보장을 위한 TSN 메커니즘의 실용적 적용 가능성을 실험적으로 입증하였으며, CBS 기반 트래픽 제어가 미래 자율주행 및 커넥티드 카 네트워크 아키텍처 설계에서 중요한 역할을 할 수 있음을 시사한다.
